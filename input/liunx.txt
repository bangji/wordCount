常见命令

chmod 777 文件名字  	权限最大
vi 文件名字  		编辑
mkdir 文件名字 		创建
ll 			详细查看
cd 目录			切换目录
cp 文件名称        	复制
mv 文件名称       	移动 重命名
rm 文件名称 		删除
cd .. 			返回上一级
tree			查看详细目录(树型)
rpm -ivh 安装包名称.rpm 	安装后缀名.rpm的软件
yum install 名称	安装
pwd			查看当前目录
cat (参数:-n,-b) 文件,文件... 查看选择部分文件内容 b=序号过滤空文件
service sshd restart
service network restart
